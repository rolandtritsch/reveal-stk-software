Course Description

Title: Having fun with Software Engineering (Hacking for Children :))

At the end of this course the children (7th - 9th) will ...

  • be able to use the keyboard and an editor (will have learned/improved their typing skills)
  • be able to go through the basic design, code, compile, run/debug cycle that you need to develop software
  • have developed a basics understanding for data structures and algorithms
  • have learned the fundamentals of a state-of-the-art programming language (e.g. Scala)

Format ...

  • 20 classes, 1 hour each
  • 15 min presentation, followed by 45 min hands-on work on the laptops
  • over the 20 classes the children will develop a project (e.g. an encoder/decoder for a cipher and (optionally) a code-breaker)

Resources ...

  • Room (for 10-15)
  • Projector
  • 10 laptops
  • I might bring a co-teacher (one of the pupils from STK)

Prerequisites ...

  • none
